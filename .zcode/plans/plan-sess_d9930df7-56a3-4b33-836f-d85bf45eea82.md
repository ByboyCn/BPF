# M6 计划:.bpfaml 标记语言 + 源生成器(完整 Avalonia 风格)

## 目标
自研 XAML 风格标记语言 `.bpfaml`,用 **Roslyn 增量源生成器** 在构建期编译成 C# 代码。
零运行时反射、NativeAOT 完全兼容。语法贴近 Avalonia .axaml,支持控件树 + 属性 + 附加属性 + 事件挂接 + code-behind(partial 类)+ x:Name + 数据绑定(编译式)+ markup extension + 资源字典 + Style。

## 核心架构决策(已与你确认)
- **解析方式**:源生成器(构建期编译),非运行时解析。README M6 既定路线。
- **范围**:完整 Avalonia 风格(控件树 + 绑定 + 资源字典 + Style + markup extension 全套)。
- **AOT 纪律**:所有生成代码用直接调用(`new T()`、`SetValue(KnownProp, ...)`、`Grid.SetRow(...)`),零反射。数据绑定改用**编译式访问器**(源生成器生成 lambda),废弃 `Binding.cs` 的反射路径。

## .bpfaml 语法设计

### 完整示例(HelloWorld 改写后)
```xml
<Window x:Class="Bpf.Samples.HelloWorld.MainWindow"
        xmlns="bpf" xmlns:x="bpf:x"
        xmlns:models="using:Bpf.Samples.HelloWorld.Models"
        Title="bpf M6 演示" Width="600" Height="500"
        x:DataType="models:MainViewModel">
  <Window.DataContext><models:MainViewModel/></Window.DataContext>
  <Window.Resources>
    <SolidColorBrush x:Key="LabelBrush" Color="#666666"/>
  </Window.Resources>
  <Grid Columns="200,*" Rows="*">
    <ScrollViewer Grid.Column="0">
      <ListBox x:Name="peopleList" ItemsSource="{Binding People}" FontSize="14"/>
    </ScrollViewer>
    <StackPanel Grid.Column="1" Orientation="Vertical">
      <Label Text="姓名:" FontSize="14"/>
      <TextBox x:Name="nameBox" Text="{Binding SelectedPerson.Name, Mode=TwoWay}"/>
      <Button Content="确定" Click="OnConfirmClick"/>
    </StackPanel>
  </Grid>
</Window>
```

### 语法元素 → 编译产物
| 元素 | 语法 | 编译产物 |
|------|------|----------|
| 对象 | `<Button/>` | `new Button()` |
| 属性 | `Content="OK"` | `btn.Content = "OK"` |
| ContentProperty | `<StackPanel><Button/></StackPanel>` | `panel.AddChild(btn)` / `border.Child=x` / `btn.Content="text"` |
| 附加属性 | `Grid.Column="1"` | `Grid.SetColumn(child, 1)` |
| 颜色 | `Background="Red"`/`"#FF8800"` | `new SolidColorBrush(Color.Parse(...))` |
| x:Name | `x:Name="myBtn"` | `private Button myBtn;` 字段 |
| x:Class | `x:Class="App.MainWindow"` | 生成 `partial class MainWindow` |
| 事件 | `Click="OnClick"` | `btn.Click += this.OnClick;`(纯 C# 绑定) |
| {Binding} | `Text="{Binding Name}"` | 编译式 lambda 绑定(需 x:DataType) |
| {StaticResource} | `Foreground="{StaticResource LabelBrush}"` | 编译期资源查找 |
| Style | `<Style TargetType=...><Setter.../>` | `new Style<T>().Add(prop, val)` |

## 分阶段实施(每阶段独立 build + 验证)

### 阶段 A:源生成器骨架 + 最小控件树
新建 `Bpf.Markup.Generator`(netstandard2.0,`IIncrementalGenerator`,读 AdditionalFiles)。
XML 解析 + 类型表(Button→`Bpf.Controls.Button`) + 代码发射(`partial class` + `InitializeComponent()`) + ContentProperty 表(Panel→AddChild/Border→Child/Button→Content)。
**验收**:HelloWorld 用 .bpfaml 显示一个含按钮的窗口。

### 阶段 B:属性转换器 + 附加属性 + x:Name
类型转换器表(string→double/int/bool/enum/Color/Brush/GridLength[])。给 `Color` 加 `Parse` 方法。
属性设置策略(StyledProperty→SetValue / CLR→直接 setter,硬编码区分表)。
附加属性 `Grid.Row/Column/RowSpan/ColumnSpan`→`Grid.SetXxx`。x:Name→生成字段。
**验收**:Grid 布局 + 多控件 + x:Name 引用。

### 阶段 C:事件挂接 + code-behind
`Click="OnClick"`→`btn.Click += this.OnClick;`(纯 C# 绑定,零反射)。
code-behind partial 类写处理方法。
**验收**:按钮/列表交互与原 Program.cs 一致。

### 阶段 D:数据绑定(编译式,AOT 安全)
新建 `CompiledBinding<TSource,TValue>` 运行时类(持有编译期 lambda)。
`Control.SetCompiledBinding` 新方法。`{Binding Path}` 要求 x:DataType,生成 `vm => vm.Person.Name` lambda。
Control 加 `DataContext` 属性 + 树继承。`<Window.DataContext>` 设根。
**验收**:列表选择→文本框自动更新(双向绑定,无手工同步)。

### 阶段 E:资源字典 + Style + markup extension
Control 加 `Resources` 字典 + 树继承查找。`<Window.Resources>` + x:Key。`{StaticResource Key}`→编译期解析。
`<Style TargetType=...><Setter.../>`→`new Style<T>().Add(...)`。`{x:Static}`/`{x:Null}`。
**验收**:Style 统一外观 + 资源复用。

### 阶段 F:测试 + AOT 验证 + 文档 + 提交
`Bpf.Markup.Generator.Tests`(CSharpGeneratorDriver 测试每语法元素)。
`dotnet publish -r win-x64 -c Release /p:PublishAot=true` 3/3 稳定。README 加 .bpfaml 章节。提交 master。

## 关键技术点

### 源生成器 csproj
```xml
<TargetFramework>netstandard2.0</TargetFramework>
<IsRoslynComponent>true</IsRoslynComponent>
<EnforceExtendedAnalyzerRules>true</EnforceExtendedAnalyzerRules>
<PackageReference Include="Microsoft.CodeAnalysis.CSharp" Version="4.11.0" PrivateAssets="all" />
```

### consumer 引用
```xml
<ProjectReference Include="...\Bpf.Markup.Generator.csproj"
                  OutputItemType="Analyzer" ReferenceOutputAssembly="false" />
<AdditionalFiles Include="**\*.bpfaml" />
```

### AOT 安全要点
- 所有生成代码是直接调用,零反射
- 绑定用编译式 lambda,废弃 `Binding(string)` 反射路径
- 资源/markup extension 编译期解析
- 不用 `MakeGenericType`/`Type.GetType(string)`
- 用 `global::` 全限定名
